# schedule-pack

Adds time-based automation primitives to OpenClaw skeletons.

## Provides

- `schedules` contract in `openclaw.json`
- `schedule-manager` skill for defining and reviewing recurring jobs
- cron-style job declarations with retry/timeout fields

## Use Cases

- hourly KPI pull and anomaly checks
- daily campaign summary and action generation
- weekly strategy refresh and budget reallocation
